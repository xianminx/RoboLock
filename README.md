RoboLock
========

android app lock sdk


# Usage


## Lock the app
call `RoboLock.onResume` and `RoboLock.onPause` in **each** Activity's `onResume` and `onPause` respectively.

## Lock Activity
call `RoboLock.onResume` and `RoboLock.onPause` in the Activity's `onResume` and `onPause` respectively.



## Lock another app
register `Receiver` to launch the LockActivity

## Manually
call `lock`
